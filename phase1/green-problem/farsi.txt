سوال سبز

به شما عدد n داده می‌شود و شما باید رشته‌ی `asm!` را با n تا s چاپ کنید. ورودی و خروجی نمونه را ببینید.

ورودی:
تنها خط ورودی  شامل n است

خروجی:
رشته‌ی مورد نظر را در یک خط چاپ کنید


محدودیت ها:
1 <= n <= 10


ورودی نمونه ۱:
```
1
```

خروجی نمونه ۱:
```
asm!
```

ورودی نمونه ۲:
```
3
```

خروجی نمونه ۲:
```
asssm!
```
